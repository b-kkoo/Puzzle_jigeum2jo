## About Game
- 퍼즐은 지금2죠
- 게임 컨셉 : 모뉴먼트 밸리(Monument Vally)를 모방, 착시 현상을 활용한 퍼즐 게임
- 조작법 : 마우스 왼쪽 클릭을 통한 입력
<br/> 이동 가능 영역 클릭 시 클릭 위치로 경로를 따라 이동 / 회전 플랫폼(bridge) 손잡이 클릭 시 플랫폼 회전

## Game Logic
- 착시 현상을 유도하는 카메라 배치 및 퍼즐 설계 / 플랫폼 회전
